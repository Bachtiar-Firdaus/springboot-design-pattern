## Ringkasan Materi

### Membuat Project
Setup project dasar untuk implementasi design patterns.

### Singleton Pattern
Konsep dasar Singleton Pattern: memastikan hanya ada satu instance dari sebuah class.

### Singleton Pattern di Spring Boot
Implementasi Singleton Pattern menggunakan Spring Boot, memanfaatkan bean scope default (`singleton`).

### Prototype Pattern
Konsep dasar Prototype Pattern: membuat objek baru dengan menyalin (clone) objek yang sudah ada.

### Prototype Pattern di Spring Boot
Implementasi Prototype Pattern di Spring Boot dengan bean scope `prototype`.

### Builder Pattern
Konsep dasar Builder Pattern: memisahkan konstruksi objek kompleks dari representasinya.

### Builder Pattern di Spring Boot
Implementasi Builder Pattern dalam Spring Boot untuk membangun objek dengan konfigurasi fleksibel.

### Factory Method Pattern
Konsep dasar Factory Method Pattern: mendefinisikan interface untuk membuat objek, tapi subclass yang menentukan kelas mana yang diinstansiasi.

### Factory Method Pattern di Spring Boot
Implementasi Factory Method Pattern di Spring Boot untuk mengelola pembuatan bean secara lebih terstruktur.

### Abstract Factory Pattern
Konsep dasar Abstract Factory Pattern: menyediakan interface untuk membuat keluarga objek terkait tanpa menentukan kelas konkret.

### Abstract Factory Pattern di Spring Boot
Implementasi Abstract Factory Pattern di Spring Boot untuk mengelola pembuatan berbagai jenis bean yang saling berhubungan.
